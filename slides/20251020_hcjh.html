🎮 做一個簡單的遊戲


# 猜數字遊戲
import random

number = random.randint(1, 10)
guess = int(input("猜一個 1-10 的數字："))

if guess == number:
    print("答對了！")
else:
    print("答案是", number)

幾行程式碼就能做出互動遊戲！
